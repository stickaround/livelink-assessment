# LiveLink Assessment

Build simple react app to see a feed of users and user detail page.

## Tech stack

- React
- TypeScript
- Material-Tailwind

## How to install

You can install dependencies using one of the below scripts.

- `npm install`
- `npm i`
- `yarn install`
- `yarn`

## How to run

You can start the app using one of the below scripts

- `npm start`
- `yarn start`
